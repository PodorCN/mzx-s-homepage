##Main Structure

PodorSystem has 3 components.
 * Podor Cloud
 * GitHub
 * Podor Host
 
 ### Podor Cloud
 PodorCloud (cloud.podor.org) is powered by a QNAP NAS with 24/7 uptime (99%), it provides,
* File Sharing
* File Syncing
* Podor DropBox

 ### Github
 * Devleopment Version Control
 
 ### Podor Host
 Podor Host is a linux server located in Toronto, Ontario
 * Website Hosting
 * Limited Storage
 * VPN Services
